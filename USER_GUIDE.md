# SmartGov Health - User Guide for New Features

## For Rural Users (తెలుగు వినియోగదారులకు)

### 1. Finding the Right Scheme Without Knowing the Name

**Problem**: "నాకు ఏ పథకం కావాలో తెలియదు"

**Solution**: Eligibility Checker
1. పథకం కార్డ్ నొక్కండి
2. "🎯 అర్హత పరీక్ష" విభాగం చూడండి
3. సాధారణ ప్రశ్నలకు "అవును" లేదా "కాదు" చెప్పండి
4. సిస్టమ్ సరిపోలే పథకం సూచిస్తుంది

### 2. Knowing Where to Go

**Problem**: "ఆసుపత్రి ఎక్కడ ఉంది?"

**Solution**: Local Help Information
1. పథకం చదవిన తర్వాత చూడండి
2. "📞 సంప్రదించండి" సెక్షన్ చూడండి
3. సమీపకు PHC, CHC, ఆసుపత్రి, village secretariat సమాచారం ఉంటుంది
4. టెలిఫోన్ నంబర్ చదవండి

### 3. Preparing Required Documents

**Problem**: "ఏ డాక్యుమెంట్‌లు తీసుకెళ్లాలి?"

**Solution**: Document Checklist
1. పథకం చూసిన తర్వాత చూడండి
2. "📋 డాక్యుమెంట్ చెక్‌లిస్ట్" కనిపిస్తుంది
3. ప్రతి డాక్యుమెంట్ కోసం ✓ చేయండి
4. office కు వెళ్లే ముందు చెక్‌లిస్ట్ ముద్రించండి (🖨️ బటన్)

### 4. Sharing with Others

**Problem**: "నా సోదరి కూడా ఈ సేవ తీసుకోవాలనుకుంటుందట"

**Solution**: WhatsApp Share
1. పథకం చూసిన తర్వాత చూడండి
2. "📱 WhatsApp" బటన్ నొక్కండి
3. సమాచారం WhatsApp లో రూపొందుతుంది
4. సోదరి, తెలిసిన వారికి పంపండి

### 5. Reading Information Aloud

**Problem**: "నేను చదవలేను"

**Solution**: Audio Support
1. పథకం చూసిన తర్వాత చూడండి
2. "🎙️ తెలుగు ఆడియో" బటన్ చూడండి
3. "▶ ప్లే" నొక్కండి
4. "🔊 మెల్లగా చదవండి" బటన్ ఇంకా నెమ్మదిగా చదువుతుంది

## For Health Workers (ఆరోగ్య సిబ్బందికి)

### Using Staff Mode (సచివాలయం సాధనాలు)

**Accessing Staff Mode**:
1. పేజీ చివరిలో "సచివాలయం / ఆరోగ్య సిబ్బంది సాధనాలు" చూడండి
2. ఆ తర్వాత నొక్కండి (▼ లో క్లిక్ చేయండి)

**Quick Scheme Lookup**:
1. డ్రాప్‌డౌన్ నుండి పథకం ఎంచుకోండి
2. "వివరాలు చూపించు" నొక్కండి
3. పథకం తెరవుతుంది

**Report Issues**:
1. "సమస్య నివేదించండి" బటన్ నొక్కండి
2. ఏ సమస్య ఎదురైందో చెప్పండి:
   - సమాచారం తప్పుగా ఉందా?
   - సంపర్క వివరాలు మార్చాలా?
3. submit చేయండి

**Add Local Contacts**:
1. "Local Details జోడించండి" నొక్కండి
2. సమీపకు health centers, offices సమాచారం జోడించండి
3. సేవ్ చేయండి

**Simplify PDF Documents**:
1. Government నుండి కొత్త డాక్యుమెంట్ మీకు వచ్చినా
2. "PDF పత్రం" విభాగంలో అప్‌లోడ్ చేయండి
3. "PDF ను సరళీకరించు" నొక్కండి
4. GEMINI AI సరళీకరిస్తుంది

## Privacy & Safety (నిరాపత్త ఉపయోగం)

### ⚠️ కూడా చేయవద్దు:

- ✅ పనిలో సహాయపడే సమాచారం సేవ్ చేయండి

## Feedback (మీ అభిప్రాయం చెప్పండి)

### Why We Ask for Feedback:
- సేవ మెరుగుపర్చుకోవటానికి
- ఎక్కెన్ని ఊరోళ్ళ ఎందుకు సేవ వ్యవహరిస్తున్నారో తెలుసుకోవటానికి
- ఎక్కెక్క సమస్యలు వస్తున్నాయో తెలుసుకోవటానికి

### How to Give Feedback:
1. పథకం చూసిన తర్వాత చూడండి
2. "👍 ఇష్టమైనది" లేదా "👎 మెరుగుపర్చండి" నొక్కండి
3. మీ సమీక్ష చేయండి
4. Submit చేయండి

### Feedback Questions:
- సమాచారం సపష్టంగా ఉందా?
- ఈ సేవ మీరు పొందారా?
- మీరు ఏ గ్రామం నుండి ఉన్నారు? (ఐచ్ఛికం)
- ఏ సమస్య ఎదురైనది?

## Troubleshooting

### Problem: ఆడియో పనిచేయటం లేదు
**Solution**: 
- Browser restart చేయండి
- Internet connection చెక్ చేయండి
- Speaker volume check చేయండి

### Problem: PDF అప్‌లోడ్ విఫలమైంది
**Solution**:
- PDF < 10 MB ఉందో చెక్ చేయండి
- .env ఫైలులో GEMINI_API_KEY సెట్ చేసిందా చెక్ చేయండి
- Internet connection చెక్ చేయండి

### Problem: తెలుగు text సరిగా కనిపిస్తుందా
**Solution**:
- Browser language నుండి Telugu నిర్ణయించటానికి
- Chrome లేదా Firefox ఉపయోగించండి
- Browser cache clear చేయండి

## Demo Scenarios

### Scenario 1: గర్భిణీ సంపాదించెట్టని ఎంటర్ అందుకోవాలనుకుంటున్నారు
1. Home screen → "జనని శిశు సురక్షా కార్యక్రమం" కార్డ్ నొక్కండి
2. Eligibility చెక్ చేస్తుంది:
   - "గర్భిణీ స్త్రీ?" → "అవును"
   - "ఆధార్ ఉందా?" → "అవును"
3. Document checklist చూడండి
4. దగ్గరకు ఆసుపత్రి నంబర్ చూడండి
5. WhatsApp ద్వారా సంబంధితికి పంపండి

### Scenario 2: ASHA Worker నిద్ర నుండి ఉదీయానికి పరిస్థితిలో ఆధోలనలను సేవ చేయడానికి
1. Home screen → "PM-JAY" నొక్కండి
2. Staff panel: "సచివాలయం సాధనాలు" విస్తరించండి
3. నిర్భయంగా సమీక్ష తెలుసుకోవటానికి నుండి
4. సమీపకు సేవల సమాచారం జోడించండి (అందుకుంటూ నవీకరించండి)
5. సమాచారం తప్పుగా ఉంటే నివేదించండి

### Scenario 3: తలుపు నుండి offline వ్యవస్థకు డేటా సేవ చేయడం
1. App సందర్శించండి (online)
2. అన్ని పథకాలు చూడండి
3. ఆడియో డౌన్‌లోడ్ చేయండి
4. జాబితా తయారీ చేయండి
5. Offline ఆ నుండి access చేయండి
