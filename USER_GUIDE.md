## Citizen Interface Guidelines (తెలుగు వినియోగదారులకు)

### 1. Scheme Identification Without Prior Knowledge
**Problem Statement**: The citizen is unaware of specific scheme nomenclature ("నాకు ఏ పథకం కావాలో తెలియదు").

**Resolution Protocol**: Eligibility Checker Initialization
1. పథకం కార్డ్ నొక్కండి
2. "🎯 అర్హత పరీక్ష" విభాగం చూడండి
3. సాధారణ ప్రశ్నలకు "అవును" లేదా "కాదు" చెప్పండి
4. సిస్టమ్ సరిపోలే పథకం సూచిస్తుంది

### 2. Geographical Location Assistance
**Problem Statement**: The citizen requires physical navigation to a service center ("ఆసుపత్రి ఎక్కడ ఉంది?").

**Resolution Protocol**: Local Help Information Retrieval
1. పథకం చదవిన తర్వాత చూడండి
2. "📞 సంప్రదించండి" సెక్షన్ చూడండి
3. సమీపకు PHC, CHC, ఆసుపత్రి, village secretariat సమాచారం ఉంటుంది
4. టెలిఫోన్ నంబర్ చదవండి

### 3. Logistical Documentation Preparation
**Problem Statement**: The citizen requires a list of mandatory documentation for office visits ("ఏ డాక్యుమెంట్‌లు తీసుకెళ్లాలి?").

**Resolution Protocol**: Document Checklist Verification
1. పథకం చూసిన తర్వాత చూడండి
2. "📋 డాక్యుమెంట్ చెక్‌లిస్ట్" కనిపిస్తుంది
3. ప్రతి డాక్యుమెంట్ కోసం ✓ చేయండి
4. office కు వెళ్లే ముందు చెక్‌లిస్ట్ ముద్రించండి (🖨️ బటన్)

### 4. Inter-community Information Dissemination
**Problem Statement**: The citizen intends to distribute scheme details within their social network ("నా సోదరి కూడా ఈ సేవ తీసుకోవాలనుకుంటుందట").

**Resolution Protocol**: WhatsApp Distribution Integration
1. పథకం చూసిన తర్వాత చూడండి
2. "📱 WhatsApp" బటన్ నొక్కండి
3. సమాచారం WhatsApp లో రూపొందుతుంది
4. సోదరి, తెలిసిన వారికి పంపండి

### 5. Auditory Consumption Modality
**Problem Statement**: The citizen requires text-to-speech assistance due to literacy limitations ("నేను చదవలేను").

**Resolution Protocol**: Audio Playback Execution
1. పథకం చూసిన తర్వాత చూడండి
2. "🎙️ తెలుగు ఆడియో" బటన్ చూడండి
3. "▶ ప్లే" నొక్కండి
4. "🔊 మెల్లగా చదవండి" బటన్ ఇంకా నెమ్మదిగా చదువుతుంది

## Administrative Field Worker Protocols (ఆరోగ్య సిబ్బందికి)

### Accessing Administrative Toolsets (సచివాలయం సాధనాలు)

**Authentication and Access**:
1. పేజీ చివరిలో "సచివాలయం / ఆరోగ్య సిబ్బంది సాధనాలు" చూడండి
2. ఆ తర్వాత నొక్కండి (▼ లో క్లిక్ చేయండి)

**Direct Scheme Query**:
1. డ్రాప్‌డౌన్ నుండి పథకం ఎంచుకోండి
2. "వివరాలు చూపించు" నొక్కండి
3. పథకం తెరవుతుంది

**Discrepancy Reporting**:
1. "సమస్య నివేదించండి" బటన్ నొక్కండి
2. ఏ సమస్య ఎదురైందో చెప్పండి:
   - సమాచారం తప్పుగా ఉందా?
   - సంపర్క వివరాలు మార్చాలా?
3. submit చేయండి

**Appending Local Coordinates**:
1. "Local Details జోడించండి" నొక్కండి
2. సమీపకు health centers, offices సమాచారం జోడించండి
3. సేవ్ చేయండి

**Automated PDF Simplification**:
1. Government నుండి కొత్త డాక్యుమెంట్ మీకు వచ్చినా
2. "PDF పత్రం" విభాగంలో అప్‌లోడ్ చేయండి
3. "PDF ను సరళీకరించు" నొక్కండి
4. GEMINI AI సరళీకరిస్తుంది

<a id="privacy-and-operational-security"></a>
## Privacy and Operational Security (నిరాపత్త ఉపయోగం)

### Restricted Operations:
- ఆధార్, ప్రిస్క్రిప్షన్స్, health documents అప్‌లోడ్ చేయవద్దు
- దీన్ని నిర్వాహించని ఉపకరణలో స్వకీయ ఫైలులను జోడించవద్దు
- సంపూర్ణ ఆధార్ నెంబర్ పంపవద్దు

### Authorized Operations:
- కేవలం పథకం సమాచారం చూడండి
- సమీపకు సేవల సమాచారం చూడండి
- పనిలో సహాయపడే సమాచారం సేవ్ చేయండి

## Offline Execution Parameters (ఇంటర్నెట్ లేనప్పుడు)

### Supported Offline Features:
- Scheme information retrieval (text and audio payloads)
- Document checklist interactions
- Emergency contact parsing (108, 104)
- Eligibility criteria evaluation
- Offline సమాచారం

### Network-Dependent Features:
- WhatsApp sharing external intents (మీకు సంపర్క అందుకుంటుంది)
- AI-driven PDF simplification (GEMINI AI చేస్తుంది)
- Dynamic repository updates (నవీకరించిన సమాచారం డౌన్‌లోడ్ చేయడం)

<a id="feedback-submission-protocol"></a>
## Feedback Submission Protocol (మీ అభిప్రాయం చెప్పండి)

### Justification for Data Collection:
- సేవ మెరుగుపర్చుకోవటానికి
- ఎక్కెన్ని ఊరోళ్ళ ఎందుకు సేవ వ్యవహరిస్తున్నారో తెలుసుకోవటానికి
- ఎక్కెక్క సమస్యలు వస్తున్నాయో తెలుసుకోవటానికి

### Submission Methodology:
1. పథకం చూసిన తర్వాత చూడండి
2. "👍 ఇష్టమైనది" లేదా "👎 మెరుగుపర్చండి" నొక్కండి
3. మీ సమీక్ష చేయండి
4. Submit చేయండి

### Standardized Interrogatives:
- సమాచారం సపష్టంగా ఉందా?
- ఈ సేవ మీరు పొందారా?
- మీరు ఏ గ్రామం నుండి ఉన్నారు? (ఐచ్ఛికం)
- ఏ సమస్య ఎదురైనది?

<a id="troubleshooting-guide"></a>
## Troubleshooting Guide

### Fault: Audio Playback Failure
**Resolution Actions**: 
- Browser restart చేయండి
- Internet connection చెక్ చేయండి
- Speaker volume check చేయండి

### Fault: PDF Ingestion Failure
**Resolution Actions**:
- PDF < 10 MB ఉందో చెక్ చేయండి
- .env ఫైలులో సక్రియ API కీ సెట్ చేసిందా చెక్ చేయండి
- Internet connection చెక్ చేయండి

### Fault: Telugu Typographical Rendering Errors
**Resolution Actions**:
- Browser language నుండి Telugu నిర్ణయించటానికి
- Chrome లేదా Firefox ఉపయోగించండి
- Browser cache clear చేయండి

## System Demonstration Scenarios

### Scenario 1: Maternal Health Benefits Evaluation
1. Home screen → "జనని శిశు సురక్షా కార్యక్రమం" కార్డ్ నొక్కండి
2. Eligibility చెక్ చేస్తుంది:
   - "గర్భిణీ స్త్రీ?" → "అవును"
   - "ఆధార్ ఉందా?" → "అవును"
3. Document checklist చూడండి
4. దగ్గరకు ఆసుపత్రి నంబర్ చూడండి
5. WhatsApp ద్వారా సంబంధితికి పంపండి

### Scenario 2: Community Health Worker Route Maintenance
1. Home screen → "PM-JAY" నొక్కండి
2. Staff panel: "సచివాలయం సాధనాలు" విస్తరించండి
3. నిర్భయంగా సమీక్ష తెలుసుకోవటానికి నుండి
4. సమీపకు సేవల సమాచారం జోడించండి (అందుకుంటూ నవీకరించండి)
5. సమాచారం తప్పుగా ఉంటే నివేదించండి

### Scenario 3: Pre-caching for Disconnected Environments
1. App సందర్శించండి (online)
2. అన్ని పథకాలు చూడండి
3. ఆడియో డౌన్‌లోడ్ చేయండి
4. జాబితా తయారీ చేయండి
5. Offline ఆ నుండి access చేయండి
